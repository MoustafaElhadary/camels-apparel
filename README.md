## Getting started

1. Clone this repository

   ```bash
   git clone https://github.com/MoustafaElhadary/intros.git
   ```

2. Install dependencies

      ```bash
      yarn
      ```
3. Create a **.env** file for the firebase config

    ```env
  
    ```
4. run 

      ```bash
      yarn dev
      ```
